# Windows-Keylogger

• Records user keystrokes (keylogging) and saves them to a file.  
• Monitors the clipboard and logs copied content.  
• Logs the title of the active window during use.  
• Takes screenshots upon mouse activity.  
• Records audio from the microphone at regular intervals.  
• Captures images from the webcam.  
• Collects system information (hostname, ipconfig).  
• Sends all data via email with file attachments.  
• Compresses logs and media into a zip file before sending.  
• Deletes old data for storage management.  
• Adds itself to startup for automatic execution.  
• Includes anti-analysis mechanisms (sandbox/VM detection and tools).  
• Terminates if it detects debugging or monitoring tools.
